# Kashmir Milk App

## 📌 Overview
Kashmir Milk is a Flutter-based mobile application designed for managing milk delivery services. It provides features such as customer management, order tracking, and local data storage using Hive and Firebase.

## ✨ Features
- 📋 **Customer Management:** Store and retrieve customer details using Hive and Firestore.
- 🚀 **Local Storage with Hive:** Offline support for storing customer data.
- 🔥 **Firebase Firestore Integration:** Sync data with Firestore for cloud storage.
- 📩 **Send Alerts:** Notify all customers with a custom message.
- 🎨 **Modern UI:** Built with Flutter for a smooth and responsive user experience.

## 🛠️ Tech Stack
- **Framework:** Flutter (Dart)
- **State Management:** Provider
- **Local Database:** Hive
- **Backend:** Firebase Firestore
- **Version Control:** Git & GitHub

## 📦 Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/yasirzain15/kashmir_milk.git
   ```
2. Navigate to the project directory:
   ```sh
   cd kashmir_milk
   ```
3. Install dependencies:
   ```sh
   flutter pub get
   ```
4. Run the app:
   ```sh
   flutter run
   ```

## 🔧 Configuration
- **Firebase Setup:**
  1. Create a Firebase project.
  2. Enable Firestore & Authentication.
  3. Download `google-services.json` and place it in `android/app/`.
